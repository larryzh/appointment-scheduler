Appointment Scheduler App

This React-based appointment scheduling app includes the following features:

  • User authentication (login, signup, password reset) using Supabase
  • Host dashboard for creating and managing appointments
  • Visitor dashboard for booking available appointments
  • Navigation between Host and Visitor dashboards
  • Route protection for authenticated users

Setup Instructions

  1. Make sure you have Node.js and npm installed on your system.
  2. Open a terminal and navigate to the project directory: cd appointment-scheduler
  3. Install the required dependencies: npm install react react-dom react-router-dom @supabase/supabase-js
  4. Start the development server: npm start

The application should now be running on http://localhost:3000

Supabase Setup

You'll need to set up the Supabase database with the following SQL script to create the necessary table for storing appointments:

create table appointments ( id uuid default uuid_generate_v4() primary key, host_id uuid references auth.users(id), visitor_id uuid references auth.users(id), date date not null, time time not null, price decimal(10, 2) not null, note text, created_at timestamp with time zone default now() );

Run this SQL in the Supabase SQL editor to create the table.

Testing the Application

You can now test the application by following these steps:

  1. Sign up as a host and create appointments
  2. Sign up as a visitor and book those appointments

The app provides a basic structure that you can further customize and expand upon as needed.
